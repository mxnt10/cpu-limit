CPU Frequency Limit Tool Fork from the GameLinux Youtube channel for Slackware.

### Required packages

- zenity >= 3.32.0

### Installation instructions

To install "cpu-limit", use the following commands:
```sh
$ git clone https://github.com/mxnt10/cpu-limit.git
$ cd cpu-limit

$ sudo ./install.sh
```
